GROUP_S-OOP Java OOP Coursework Project Module Object Oriented Programming

Description This repository contains solutions for the Java OOP coursework, covering both console-based and GUI-based applications as specified in the coursework questions.

Question One Part A: Students Grader A console-based Java application that: Accepts scores for five students using a while loop Determines grades and remarks based on the Uganda O-Level grading system The system shows student test results alongside their complete academic performance records.

Part B: SK Motors Auction System A console-based Java application that: The system allows three bidders to participate before it selects the bidder who submitted the highest price. The system records all vehicle information together with payment records and financial records and monetary transactions. Computes and displays profit or loss

Question Two Student Registration System (GUI) A Java Swing desktop application that: Replicates a student registration form The system performs a complete verification process to check all input data. The system produces a student information document which follows a specific structure. Saves records to a CSV file (students.csv) as specified in the coursework instructions Notes The project strictly follows the requirements stated in the coursework questions. Data persistence for Question Two is implemented using CSV as required.
